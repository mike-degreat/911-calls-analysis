# 911 Calls Data Analysis

This project analyzes emergency 911 call data using:
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Key Insights
- EMS had the highest call volume
- Peak emergency activity occurred around 5 PM
- LOWER MERION had the highest township call volume
- Emergency calls showed seasonal fluctuations

## Tools Used
- Python
- Jupyter Notebook
- Seaborn
- Matplotlib
